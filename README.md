# MotoGp-Plotting
Matplotlib used to graph the MotoGp World Championship.

## Python version
- Python 3.9.6 64-bit

## Python script library dependencies
- matplotlib==3.5.1

## How to run
From base repository run:

```
python .\src\general-world-championship.py
```
